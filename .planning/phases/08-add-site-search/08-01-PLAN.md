<task type="auto">
  <name>Task 2: Add Pagefind indexing attributes to all page templates</name>
  <files>src/pages/[slug].astro, src/pages/index.astro, src/pages/blog/[...page].astro, src/pages/category/[slug].astro, src/pages/videos/[slug].astro, src/pages/videos/index.astro</files>
  <action>
    Add `data-pagefind-body` to the `<main>` element on each page template so only article content is indexed. Add `data-pagefind-ignore="all"` to sections that should NOT be indexed.

    **src/pages/[slug].astro (blog posts and pages):**
    - Both `<main>` elements (blog branch and page branch): add `data-pagefind-body`
    - On the blog branch, add `data-pagefind-ignore="all"` to:
      - The Giscus component wrapper: wrap `<Giscus />` in `<div data-pagefind-ignore="all">` (line ~163)
      - The related posts section: add `data-pagefind-ignore="all"` to the related posts wrapper div (line ~166)
      - The Subscribe component wrapper: wrap `<Subscribe .../>` in `<div data-pagefind-ignore="all">` (line ~196)
      - The share links div (line ~135): add `data-pagefind-ignore="all"`
    - On blog posts, add `data-pagefind-filter="category"` to each category link. The categories are rendered as `<a>` tags inside the categories div. Add the attribute to the `<a>` element that displays each category name.

    **src/pages/index.astro (homepage):**
    - Add `data-pagefind-body` to `<main>`
    - The homepage is mostly post previews -- this is fine, it will index the homepage content

    **src/pages/blog/[...page].astro (blog listing):**
    - Do NOT add `data-pagefind-body` to this page -- listing pages with post previews would create duplicate index entries. Instead, add `data-pagefind-ignore="all"` to the `<main>` element.

    **src/pages/category/[slug].astro (category archives):**
    - Same as blog listing -- add `data-pagefind-ignore="all"` to `<main>` to avoid duplicate indexing of post previews.

    **src/pages/videos/[slug].astro (individual video pages):**
    - Add `data-pagefind-body` to `<main>`

    **src/pages/videos/index.astro (videos hub):**
    - Add `data-pagefind-ignore="all"` to `<main>` to avoid duplicate indexing.
  </action>
