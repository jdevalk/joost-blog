# **WordPress plugins**

I was the original author of and still sometimes contribute code to [Yoast SEO](https://yoast.com/wordpress/plugins/seo/) and its addons. Currently, I focus mostly on [Progress Planner](https://progressplanner.com/). Next to that, I maintain a few simple WordPress plugins myself:

## Comment Experience

WordPress’ comment system is incomplete. This plugin adds a lot of nifty little tricks to make comments on WordPress a lot more manageable.

[Comment Experience](https://progressplanner.com/plugins/comment-experience/)

## XML sitemaps for PDFs

If your site has a lot of PDFs, this add-on to Yoast SEO is what you need to get them indexed.

[XML sitemaps for pdfs](/plugins/pdf-xml-sitemap/)

## Clicky Analytics

Make it easy for you to add your [Clicky analytics](http://clicky.com/145844) tracking code to your site.

[Clicky analytics](/plugins/clicky/)

##   Fewer tags

Make sure tags are only live on your site when they actually add value, so when there’s more than a few posts in it. This makes sure your site has less (useless) tags.

[Fewer tags](https://progressplanner.com/plugins/fewer-tags/)
